# 🤖 CodeDoctor AI

🚀 CodeDoctor AI is a simple web app that helps developers understand and fix coding errors instantly.

---

## 🔥 Features

- 🧠 Detects common coding errors
- 💡 Suggests solutions
- 🧾 Provides fixed code
- 📋 Copy code button
- 🎨 ChatGPT-like UI

---

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express

---

## 🚀 How to Run

### Backend
```bash
cd backend
npm install
node server.js
